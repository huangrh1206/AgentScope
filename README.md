# 🎮 三国狼人杀 (Three Kingdoms Werewolf)

基于 **AgentScope** 框架开发的中文版狼人杀游戏。本项目创造性地将《三国演义》中的经典角色（如刘备、曹操、诸葛亮等）与传统狼人杀玩法相结合，利用大语言模型（LLM）的推理能力实现高度智能的角色扮演和策略对抗。

---

## ✨ 核心特性

- **🎭 三国身份深度融合**：每位玩家不仅拥有狼人杀角色，还具备三国角色的性格特点。关羽的忠义、诸葛亮的深谋远虑都会影响其发言风格。
- **🤖 智能 Agent 架构**：基于 `ReActAgent` 和结构化输出（Structured Output），Agent 能够进行逻辑推理、秘密讨论、查验目标及精准杀人。
- **💬 封闭式消息广播**：利用 AgentScope 的 `MsgHub` 机制，实现了完美的阵营通信。例如：狼人仅在夜晚拥有专属的讨论频道，好人对此完全不可见。
- **📝 实时 Markdown 日志**：自研 `GameLogger` 模块，实时劫持终端输出并将其转化为排版精美的 Markdown 游戏战报，支持实时追溯。
- **🛠️ 结构化交互协议**：通过 Pydantic 模型强制模型生成结构化数据，确保投票、行动理由和逻辑分析的严谨性和稳定性。
- **⚡ 灵活适配**：支持 6 人、8 人及 9 人标准场配置。

## 🚀 快速开始

### 1. 环境准备
确保你已安装 Python 3.9+。建议使用 `conda` 管理环境：
```bash
conda create -n env_agentscope python=3.10
conda activate env_agentscope
pip install agentscope pydantic python-dotenv
```

### 2. 配置 API Key
在项目根目录下创建 `.env` 文件，并填写你的 API 信息：
```env
AIHubMix_API_KEY="你的_API_KEY"
```
*注：本项目默认使用 `coding-glm-5-free` 模型，你也可以在 `main_cn.py` 中自行更改模型配置。*

### 3. 运行游戏
```bash
python main_cn.py
```

## 🎮 游戏玩法

1.  **准备阶段**：系统根据配置（默认 6-9 人）自动为所有玩家分配三国角色及狼人杀身份。
2.  **夜晚开始**：
    *   **狼人杀人**：狼人在 `MsgHub` 中进行秘密讨论并达成共识选择击杀目标。
    *   **预言家验证**：查验一名玩家，得知其是否为狼人。
    *   **女巫行动**：决定是否使用解药救活死者，或使用毒药毒杀他人。
3.  **白天揭晓**：主持人公布死讯，存活玩家按照顺序进行发言讨论。
4.  **公投投票**：所有存活玩家投出最有嫌疑的人，并执行淘汰。
5.  **判断胜负**：当一方阵营被全灭，或狼人数量达到总人数一半时游戏结束。

## 🛠️ 技术细节

### 📜 实时日志系统
为了解决终端内容跳动快、难以回溯的问题，本项目引入了 `GameLogger`：
- **实时性**：每一行 `print()` 内容都会同步写入磁盘。
- **可视化**：自动将角色分配、投票结果生成 Markdown 表格，并为关键游戏循环添加标题和带时间戳的引用（Events）。

### 🧠 智能提示词
通过 `ChinesePrompts` 类，我们为 Agent 注入了多层级的约束：
1.  **系统级约定**：强制遵循 JSON 格式输出。
2.  **角色级背景**：如“你是张飞，说话大声直接，容易冲动”。
3.  **规则级引导**：明确目标阵营及技能使用方法。
